BUT WAIT!

After deployment, style sheets will be broken because the Maian Music app doesn't know its own URL.

* Browse to the URL where you deployed Maian Music and add /admin/index.php?p=settings` to the end of the path
* Scroll down to Store HTTP Path and enter the fully qualified path to your site, taking care to specify http:// or https:// as appropriate
* Set Server Path to Secure Folder as /var/www/secure
* Click Update

You can now begin using Maian Music.
